# 📚 Book Library App

A simple and responsive **Book Library web application** built with **React + Vite** and **Tailwind CSS**.  
Users can add, edit, delete and manage their books easily. All data is stored in **LocalStorage**, so it persists even after page refresh.

---

## 🚀 Features

- ➕ Add new books
- 📋 List all books
- ✏️ Edit existing books
- 🗑️ Delete books
- 💾 Data persistence with LocalStorage
- 🔍 Simple and clean UI
- 📱 Responsive design

---

## 🛠️ Tech Stack

- React (Vite)
- Tailwind CSS
- JavaScript (ES6+)
- LocalStorage API
